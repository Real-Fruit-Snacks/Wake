01 Premise

Tasks are data. Notes are prose.

Wake is an Obsidian plugin that maintains a self-contained task database. Tasks are stored in the plugin's data.json — Wake does not scan, parse, or modify your markdown notes.

If you want tasks embedded in markdown, you want Tasks or Dataview. They're great. Wake is the opposite philosophy: vault stays clean, tasks stay structured, history stays forever.

02 Specs

What's in the box.

STORAGE
Self-contained data.json — no markdown parsing, no vault clutter.
VIEWS
6 smart lists — Active · Today · Overdue · This Week · No Date · Logbook.
PROJECTS
Color-tagged · archive · drag-to-reorder · or none (Inbox).
RECURRENCE
every: DSL — every:day, every:monday, every:3-days.
PALETTE
Ctrl+K for every action · Ctrl+Shift+T quick-add with live preview.
STACK
Pure TypeScript · Obsidian Plugin API · zero external dependencies.
03 Quickstart

Install, open, type.

Manual install or via BRAT. After enabling, click the check-circle ribbon or run Open Wake panel from the command palette.

# Manual install
1. Download manifest.json, main.js, styles.css # from latest release
2. Create <your-vault>/.obsidian/plugins/wake/
3. Drop the three files into that folder
4. Enable Settings → Community plugins → Wake
5. Open ribbon icon, or "Open Wake panel" from palette

# Quick-add — Ctrl+Shift+T
"Buy groceries due:tomorrow #errands [[Shopping list]] !2"
                              # All metadata in one line
04 Syntax

Quick-add tokens.

today / tomorrow
Due date shortcuts
due:YYYY-MM-DD
Explicit due date
start:YYYY-MM-DD

every:VALUE
day, week, monday, 3-days
#tag
Cross-cutting tag (multiple allowed)
[[Note]]
Link to a vault note (clickable)
!1 ... !4
Priority — urgent → low

06 Philosophy

Vault stays clean. Database stays queryable.

Obsidian-native task plugins almost universally scan and parse your markdown. That works until you want priorities, recurrence, structured projects, completion history, or smart lists across hundreds of notes. Then the parser fights you, the syntax bloats, and your vault becomes a database masquerading as prose.

Wake takes the opposite stance: tasks are data, notes are prose. Tasks reference notes via [[Note]] links, but the data lives separately and structured.

No emojis anywhere. Clean typography throughout.
